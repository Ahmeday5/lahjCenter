# LahjCenter Dashboard

## Description
LahjCenter Dashboard is an **Angular 18 admin panel** for managing the LahjCenter platform.  
Through this dashboard, administrators can fully control ads, user comments, users, categories, services, and regions, with detailed reporting capabilities.

### Key Features

- **Advertisements Management**
  - View, accept, reject, edit, or delete ads
  - Manage ad categories, services, and regions
  - Full CRUD for ads

- **Comments Management**
  - Approve or reject comments before they appear
  - Delete inappropriate comments
  - View reports on comments
  - Block users who violate rules
  - Full CRUD for comments

- **User Management**
  - View and manage all users
  - Add, edit, delete users
  - Assign roles and permissions

- **Categories & Services**
  - Manage advertisement categories
  - Manage services and transactions
  - Manage regions/governorates
  - Full CRUD functionality

- **Reports & Analytics**
  - Track platform activity
  - Generate reports for ads, users, and transactions
  - Insights on user engagement and system performance

### Technologies Used
- Angular 18
- TypeScript
- HTML / CSS / SCSS
- RxJS

### Notes
- This repository contains the **UI and code structure** for the dashboard.  
- Any live data (ads, comments, users) are **connected to the real system** and are not included in this repo for security.  
- Safe to clone or review for learning, testing, or portfolio purposes.
